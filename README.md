# Animagem

Marketplace de ultrassonografia veterinária domiciliar — conecta veterinários ultrassonografistas autônomos a tutores de pets.

- **Frontend:** `public/` — publicado no Cloudflare Pages (https://animagem.pages.dev)
- **Banco/Auth:** Supabase (Postgres + Row Level Security) — schema em [`supabase/migrations`](supabase/migrations)
- **Docs e contexto do projeto:** [`docs/`](docs)
- Setup inicial: [`docs/SETUP.md`](docs/SETUP.md)
